Ghostwriter

Enhanced bookwork mod. It features:
  • Copying and pasting individual pages, groups of pages, and entire books
  • Loading from and saving to human-readable files on your hard drive
  • Live editing books in an external text editor
  • Buttons to insert formatting characters

Quiet Time

Chat suppression mod. Features:
  • Toggle on/off the new message popups. Messages will still be added to the history, so you can see them when you press T
  • Convert snitch co-ordinates to hovertext
  • Snitch message rate limiting, (only see one snitch message per user every X seconds)
